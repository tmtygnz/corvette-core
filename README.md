## Corvette Core
A blazingly fast and robust Network Video Recorder (NVR) for residential and small business use.

Features:

- [x] Camera Streaming & Recording
- [x] Camera AI Detection
- [ ] HLS playlist for video playback
- [ ] Livestreaming
- [ ] Events Logging
- [ ] Web UI Support
- [ ] ONVIF Support
